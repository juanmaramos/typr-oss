## Default Permission

Default permissions for the config plugin - allows all config operations

#### This default permission set includes the following:

- `allow-get-general-config`
- `allow-set-general-config`
- `allow-get-notification-config`
- `allow-set-notification-config`
- `allow-get-ai-config`
- `allow-set-ai-config`
